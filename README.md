# Jogo Pin-Pan em Go

Este projeto é um simples programa em Go que implementa o jogo "Pin-Pan". Este jogo é uma brincadeira popular que substitui números por palavras dependendo de suas propriedades de divisibilidade.

## Descrição

O jogo percorre os números de 1 a 100 e substitui:
- Números divisíveis por 3 por "Pin".
- Números divisíveis por 5 por "Pan".
- Números divisíveis por ambos 3 e 5 por "PinPan".

Os números que não são divisíveis nem por 3 nem por 5 são exibidos normalmente.

## Como Executar

Para rodar este programa, você precisa ter o Go instalado em seu computador. Aqui estão os passos para executar o programa:

1. Clone este repositório.
2. Abra um terminal.
3. Navegue até o diretório onde o arquivo está salvo.
4. Execute o comando `go run main.go`.
